Homework: Text Files
 1. Odd lines

 a  that reads a text file and prints on the console its odd lines.
 2. Concatenate text files

 a  that concatenates two text files into another text file.
 3. Line numbers

 a  that reads a text file and inserts line numbers in front of each of its lines.
The result should be written to another text file.
 4. Compare text files

 a  that compares two text files line by line and prints the number of lines that are the same and the number of lines that are different.
Assume the files have equal number of lines.
 5. Maximal area sum

 a  that reads a text file containing a square matrix of numbers.
Find an area of size 2 x 2 in the matrix, with a maximal sum of its elements.
The first line in the input file contains the size of matrix N.
Each of the next N lines contain N numbers separated by space.
The output should be a single number in a separate text file.
